Notre trio de robots veut jouer au golf, mais ce sont les rois du green et ils manient le club comme des dieux : la balle est dans le trou en un coup. Alors ils ont changé les règles du jeu pour pimenter un peu les parties. Maintenant, ils doivent mettre la balle dans les trous dans un ordre spécial.

Il y a cinq (5) trous sur le green carré de taille 10 x 10 unités. On vous donne les coordonnées des trous du parcours. Pour gagner, il faut trouver le parcours le plus court pour mettre la balle dans tous le trous, et votre fonction doit retourner la longueur de ce parcours. Le parcours commence au point (0, 0) et peut se terminer par n'importe quel trou. Le résultat sera testé avec une précision de ±0.01.

Pour cette mission, l'objectif est de faire un code le plus court possible. Plus votre code est court, plus vous gagnez de points. Votre score est calculé en fonction de la longueur de votre code (nombre de carcatères): un code de 350 caractères ne rapporte aucun point ; chaque caractère de moins que 350 vous rapporte 1 point. Par exemple, un code de 300 caractères rapporte 50 points.

Input: Un ensemble (set) des coordonnées des trous. Chaque trou est représenté par ses coordonnées, comme un couple (tuple) d'entiers (int).

Output: La longueur du plus court parcours (float).

Exemple: 

golf({(2, 2), (2, 8), (8, 8), (8, 2), (5, 5)}) == 23.31
golf({(2, 2), (4, 4), (6, 6), (8, 8), (9, 9)}) == 12.73

Application: Il ne s'agit pas d'élégance ou de performance dans cette mision, mais des trucs et astuces de la programmation en Python, qui aident à faire un code le plus court possible. Cela peut aider à meilleure compréhension du fonctionnement de Python, bien que l'on ne code pas comme cela au quotidien.

Préconditions: len(holes) == 5
all(0 < x < 10 and 0 < y < 10 for x, y in holes)
